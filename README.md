## 📝 프로젝트 소개

**Calorie-chat**은 사용자의 키와 몸무게를 바탕으로 일일 섭취 칼로리를 계산하고,  
입력된 식단 정보를 기반으로 **생성형 AI**가 영양 피드백을 제공하는 서비스입니다.

### 🚀 핵심 기능
- JWT 기반 로그인 / 회원가입
- 사용자 프로필(키, 몸무게, 활동 수준) 등록
- 일일 목표 칼로리 자동 계산
- 식단 등록 및 조회
- 식단에 대한 GPT 기반 AI 피드백 제공
- AI 분석 레포트 히스토리 확인 (달력 기반)
