## 📝 소개
사용자 몸무게와 키를 바탕으로 한 일일 목표칼로리 계산 및 생성형 AI를 활용한 식단 피드백

기능(간단)
- 로그인/회원가입
- 사용자 정보 입력
- 일일 섭취 목표 칼로리 계산
- 식단 등록
- 식단에 대한 AI분석
- AI 분석 레포트
